- Charter, governance, compatibility, and FIP templates.
- Complete initial FIP portfolio.
- CI policies for independent build constraints.

- Audited seed artifact contract.

- `fin-seed -> finc -> finc` deterministic closure.

## M4 Windows PE

- Direct PE emitter and runtime parity.

## M5 finobj + finld

- Minimal object format and linker.

## M6 Ecosystem and Performance

- Registry flow, diagnostics improvements, and optimization work.
